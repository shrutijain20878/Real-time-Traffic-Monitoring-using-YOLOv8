# AI Smart Traffic Monitoring System

YOLOv8 based traffic monitoring system with vehicle detection,
tracking, speed estimation and violation detection.

Features

• Vehicle detection using YOLOv8
• Multi-object tracking with DeepSORT
• Vehicle counting using virtual line
• Speed estimation
• Speed violation detection

Tech Stack

Python  
YOLOv8  
DeepSORT  
OpenCV  
PyTorch  

Run

pip install -r requirements.txt

python src/detect_video.py
